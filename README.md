# The Jutty's

### The Jutty's is a blog management system based on Spring Boot 

environment information (tech stack):
* Gradle

* SpringBoot 

* Spring Security

* Thymeleaf

* MySQL 5.7+

* Elasticsearch 2.4.4 

* MongoDB file server  
https://github.com/waylau/mongodb-file-server

how to run:

* `./gradlew bootRun`
